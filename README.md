# 🏥 Pharmacy Sales Analysis

## 📌 Objective
Analyze sales and profit data of a pharmacy.

## 🛠 Tools Used
- Excel
- Power BI

## 📊 Key Insights
- Tablets generate highest revenue  
- Chennai region has highest sales  
- UPI is most used payment method  

## 🚀 Recommendations
- Focus on high-demand products  
- Improve low-performing regions  
- Promote high-margin products
